# Brief Execution Rule

Use this rule for commentary and progress updates while a skill is executing.

Rules:

- Prefer one short sentence.
- If the update can stay under 17 words and still be clear, use the shorter version.
- Say only current status, next step, or blocker.
- Do not explain obvious actions while the skill is actively executing.
- Keep final answers separate; this rule is mainly for execution-time updates.

Examples:

- `Đang poll Claude, chưa có result.`
- `Build pass, đang verify UI.`
- `Đã xong SOW_0029, đang commit.`
